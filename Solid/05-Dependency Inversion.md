**Dependency Inversion Principle (DIP)** рЃљрЃарЃўрЃА SOLID рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃљ, рЃарЃбрЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЊрЃўрЃА рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃЮрЃЉрЃљрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА.

рЃћрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃБ рЃњрЃЋрЃЎрЃљрЃарЃюрЃљрЃ«рЃЮрЃЉрЃА рЃЮрЃа рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃгрЃћрЃАрЃА:
1. **рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў (High-Level Modules) рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЋрЃюрЃћрЃю рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃќрЃћ (Low-Level Modules). рЃЮрЃарЃўрЃЋрЃћ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃБрЃЊрЃћрЃЉрЃБрЃџрЃў рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљрЃќрЃћ.**
2. **рЃљрЃЉрЃАрЃбрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃќрЃћ, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃћрЃЉрЃќрЃћ.

рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў, **рЃљрЃа рЃБрЃюрЃЊрЃљ рЃњрЃЋрЃЦрЃЮрЃюрЃЊрЃћрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃќрЃћ,** рЃљрЃарЃљрЃЏрЃћрЃЊ рЃўрЃАрЃўрЃюрЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃфрЃћрЃЌ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

## 1. DIP -рЃўрЃА рЃЊрЃљрЃарЃдрЃЋрЃћрЃЋрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

рЃЊрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ,рЃарЃЮрЃЏ рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф **рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљрЃА** рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА.
```php
class EmailService {
    public function sendEmail(string $to, string $message) {
        echo "Sending email to $to: $message\n";
    }
}

class UserService {
    private EmailService $emailService;

    public function __construct() {
        $this->emailService = new EmailService(); // РЮї DIP рЃЊрЃљрЃўрЃарЃдрЃЋрЃљ!
    }

    public function registerUser(string $email) {
        echo "User registered: $email\n";
        $this->emailService->sendEmail($email, "Welcome to our platform!");
    }
}

// рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
$userService = new UserService();
$userService->registerUser("user@example.com");
```

Рџа№ИЈ **рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў:**

- **рЃЏрЃГрЃўрЃЊрЃарЃЮ рЃЎрЃљрЃЋрЃерЃўрЃарЃў (Tight Coupling):** UserService **рЃЏрЃГрЃўрЃЊрЃарЃЮрЃЊ рЃљрЃарЃўрЃА рЃЏрЃўрЃЉрЃЏрЃБрЃџрЃў** EmailService-рЃќрЃћ, рЃарЃљрЃф рЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА (SMS, Push Notification) рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљрЃА.
- **рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃарЃЌрЃБрЃџрЃћ:** рЃЌрЃБ EmailService-рЃерЃў рЃарЃљрЃЏрЃћ рЃерЃћрЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ, рЃћрЃА рЃњрЃљрЃ«рЃЊрЃћрЃЉрЃљрЃА рЃЏрЃЮрЃљрЃ«рЃЊрЃћрЃюрЃА UserService-рЃќрЃћрЃф.
- **рЃарЃЌрЃБрЃџрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ:** Unit Testing рЃЊрЃарЃЮрЃА рЃарЃЌрЃБрЃџрЃўрЃљ UserService-рЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ EmailService рЃЎрЃџрЃљрЃАрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

## 2. DIP-рЃўрЃА рЃЊрЃљрЃфрЃЋрЃљ (Dependency Injection-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ)

**рЃАрЃгрЃЮрЃарЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљрЃљ** UserService рЃљрЃа рЃўрЃДрЃЮрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ EmailService рЃЎрЃџрЃљрЃАрЃќрЃћ, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃўрЃЏрЃБрЃерЃљрЃЮрЃА **рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА (Interface) рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ.**

```php
interface NotificationService {
    public function send(string $to, string $message);
}
```

рЃљрЃ«рЃџрЃљ, **EmailService** рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА рЃњрЃљрЃБрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃљрЃЏ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА:

```php
class EmailService implements NotificationService {
    public function send(string $to, string $message) {
        echo "Sending email to $to: $message\n";
    }
}
```

UserService рЃљрЃдрЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ EmailService-рЃќрЃћ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў, рЃљрЃарЃљрЃЏрЃћрЃЊ **рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљрЃќрЃћ (NotificationService).**

```php
class UserService {
    private NotificationService $notificationService;

    public function __construct(NotificationService $notificationService) {
        $this->notificationService = $notificationService;
    }

    public function registerUser(string $email) {
        echo "User registered: $email\n";
        $this->notificationService->send($email, "Welcome to our platform!");
    }
}
```

рЃљрЃ«рЃџрЃљ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ **SMS** рЃЊрЃљ **Push Notification:**

```php
class SmsService implements NotificationService {
    public function send(string $to, string $message) {
        echo "Sending SMS to $to: $message\n";
    }
}
```

## 3. DIP - рЃўрЃА рЃАрЃљрЃарЃњрЃћрЃЉрЃћрЃџрЃў

**РюЁ Loose Coupling - рЃЏрЃЮрЃ«рЃАрЃюрЃўрЃџрЃўрЃљ рЃЏрЃГрЃўрЃЊрЃарЃЮ рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў**

рЃЎрЃЮрЃЊрЃў рЃБрЃцрЃарЃЮ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃюрЃљрЃф UserService рЃљрЃа рЃљрЃарЃўрЃА рЃЏрЃўрЃЉрЃЏрЃБрЃџрЃў рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ EmailService - рЃќрЃћ.

```php
// Email рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ
$emailService = new EmailService();
$userService = new UserService($emailService);
$userService->registerUser("user@example.com");

// SMS рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ
$smsService = new SmsService();
$userService = new UserService($smsService);
$userService->registerUser("user@example.com");
```



**РюЁ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ**

рЃЌрЃБ рЃЏрЃЮрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃњрЃЋрЃГрЃўрЃарЃЊрЃљ **Push Notification** рЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџ рЃЎрЃџрЃљрЃАрЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЦрЃюрЃћрЃЉрЃљ NotificationService рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ.

```php
class PushNotificationService implements NotificationService {
    public function send(string $to, string $message) {
        echo "Sending push notification to $to: $message\n";
    }
}
```
UserService рЃерЃў рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃљрЃа рЃЊрЃљрЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ!


**РюЁ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ**

рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ **Mock рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў** рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

```php
class MockNotificationService implements NotificationService {
    public function send(string $to, string $message) {
        echo "[Mock] Notification sent to $to: $message\n";
    }
}

$mockService = new MockNotificationService();
$userService = new UserService($mockService);
$userService->registerUser("test@example.com");
```

## 4.  DIP рЃЊрЃљ Dependency Injection (DI)

**Dependency Inversion** рЃ«рЃерЃўрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА **Dependency Injection**-рЃЌрЃљрЃю, рЃарЃљрЃЊрЃњрЃљрЃю DI-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЎрЃџрЃљрЃАрЃА рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃбрЃЮрЃарЃўрЃА рЃљрЃю рЃАрЃћрЃбрЃћрЃарЃўрЃЌ рЃЏрЃўрЃћрЃгрЃЮрЃЊрЃћрЃЉрЃљ.

рЃЉрЃћрЃЋрЃа PHP **Framework** рЃерЃў (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, **Laravel,  Symfony**) **Service Contrainer** рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ DIP-рЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃљрЃА:

```php
// Laravel-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
app()->bind(NotificationService::class, EmailService::class);

$userService = app(UserService::class);
$userService->registerUser("user@example.com");
```

**­Ъџђ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ**

**РюЁ Dependency Inversion Principle (DIP) рЃљрЃарЃўрЃА рЃЎрЃљрЃарЃњрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю:**

1. **рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃЏрЃГрЃўрЃЊрЃарЃЮ рЃЎрЃљрЃЋрЃерЃўрЃарЃА (Tight Coupling)** рЃЊрЃљ рЃ«рЃЊрЃўрЃА рЃЎрЃЮрЃЊрЃА рЃБрЃцрЃарЃЮ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃА.
2. **рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЕрЃљрЃЋрЃљрЃюрЃљрЃфрЃЋрЃџрЃЮрЃЌ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў** - рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ SmsService, EmailService, PushNotificationService - UserService рЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.
3. **рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ:** рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ Mock рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.
4. **рЃЎрЃЮрЃЊрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃБрЃцрЃарЃЮ рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў.**

**рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ:**

- рЃарЃЮрЃфрЃљ рЃЎрЃџрЃљрЃАрЃў рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃАрЃћрЃарЃЋрЃўрЃАрЃќрЃћ рЃЊрЃљрЃњрЃЋрЃўрЃюрЃЊрЃљ рЃўрЃА рЃБрЃцрЃарЃЮ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃњрЃљрЃЋрЃ«рЃљрЃЊрЃЮрЃЌ.
- рЃарЃЮрЃфрЃљ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃЌ.
- рЃарЃЮрЃфрЃљ рЃњрЃЋрЃўрЃюрЃЊрЃљ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃЮрЃЉрЃљ.
