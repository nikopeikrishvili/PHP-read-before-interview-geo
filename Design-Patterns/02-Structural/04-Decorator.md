**Decorator Pattern** рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃю рЃърЃљрЃбрЃћрЃарЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЎрЃџрЃљрЃАрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃљрЃа рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃЮрЃА рЃЏрЃўрЃАрЃў рЃЌрЃљрЃЋрЃЊрЃљрЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЎрЃЮрЃЊрЃўрЃА, рЃћрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА **рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЦрЃфрЃћрЃЋрЃћрЃЉрЃў рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃљрЃЊ** рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА.

PHP-рЃерЃў **Decorator Pattern** рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃњрЃЋрЃАрЃБрЃарЃА рЃЎрЃџрЃљрЃАрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ рЃЏрЃўрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃћрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў рЃўрЃАрЃћрЃЌрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃўрЃЋрЃА рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ, рЃЦрЃћрЃерЃўрЃарЃћрЃЉрЃљ рЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃњрЃљрЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ.

---

#### Decorator Pattern - рЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
1. **Component (рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў)** - рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџрЃў рЃЎрЃџрЃљрЃАрЃўрЃА рЃљрЃю рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў.
2. **Concrete Component (рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў)** - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃГрЃўрЃарЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃерЃћрЃўрЃфрЃљрЃЋрЃА.
3. **Decorator (рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃў)** - рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџрЃў рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃА рЃљрЃЋрЃАрЃћрЃЉрЃА рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ.
4. **Concrete Decorator (рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃў)** - рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃарЃћрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃБрЃЏрЃљрЃљрЃбрЃћрЃЉрЃА.
---

#### Decorator Pattern рЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ
рЃЏрЃЮрЃЊрЃўрЃЌ, рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃЮрЃЌ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў - рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ, рЃАрЃљрЃЊрЃљрЃф рЃњрЃЋрЃљрЃЦрЃЋрЃА **TextFormatter** рЃЊрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃбрЃћрЃЦрЃАрЃбрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃбрЃўрЃџрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃљрЃю.

##### 1. рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў (Component)

```php
<?php

interface TextFormatter {
    public function format(string $text): string;
}
```
рЃћрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃА **format()**, рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃџрЃљрЃАрЃўрЃА рЃЏрЃўрЃћрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў.

##### 2. рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃџрЃљрЃАрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ (Concrete Class)

```php
class PlainText implements TextFormatter {
    public function format(string $text): string {
        return $text;
    }
}
```

рЃћрЃА рЃЎрЃ»рЃџрЃљрЃАрЃў рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃА рЃўрЃАрЃћрЃЌрЃљрЃЊ, рЃарЃЮрЃњрЃЮрЃарЃўрЃф рЃљрЃарЃўрЃА.

##### 3. рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ (Base Decorator)

```php
class TextDecorator implements TextFormatter {
    protected TextFormatter $formatter;

    public function __construct(TextFormatter $formatter) {
        $this->formatter = $formatter;
    }

    public function format(string $text): string {
        return $this->formatter->format($text);
    }
}
```

рЃљрЃарЃћ рЃљрЃарЃўрЃА **рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџрЃў рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃў**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЏрЃЊрЃњрЃЮрЃЏрЃерЃў рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ.

##### 4. рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

**UppercaseDecorator - рЃбрЃћрЃЦрЃАрЃбрЃЮрЃА рЃљрЃАрЃЮрЃћрЃЉрЃўрЃА рЃЏрЃљрЃдрЃљрЃџ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃерЃў рЃњрЃљрЃЊрЃљрЃДрЃЋрЃљрЃюрЃљ:**

```php
class UppercaseDecorator extends TextDecorator {
    public function format(string $text): string {
        return strtoupper($this->formatter->format($text));
    }
}
```

**HTMLBoldDecorator - рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃбрЃћрЃњрЃўрЃЌ рЃњрЃљрЃцрЃЮрЃарЃЏрЃћрЃЉрЃљ:**

```php
class HTMLBoldDecorator extends TextDecorator {
    public function format(string $text): string {
        return '<b>' . $this->formatter->format($text) . '</b>';
    }
}
```


##### 5. рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

```php
$text = new PlainText();
echo $text->format("Hello, World!"); 
// рЃерЃћрЃЊрЃћрЃњрЃў: Hello, World!

$uppercaseText = new UppercaseDecorator($text);
echo $uppercaseText->format("Hello, World!"); 
// рЃерЃћрЃЊрЃћрЃњрЃў: HELLO, WORLD!

$boldText = new HTMLBoldDecorator($uppercaseText);
echo $boldText->format("Hello, World!"); 
// рЃерЃћрЃЊрЃћрЃњрЃў: <b>HELLO, WORLD!</b>
```


#### Decorator Pattern - рЃўрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў

РюЁ **рЃЎрЃЮрЃЊрЃерЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ** - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ рЃЎрЃџрЃљрЃАрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ.    
РюЁ **рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃА рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃўрЃарЃћрЃЉрЃљ** - рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃљрЃЊ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮрЃЌ рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃЊрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў.    
РюЁ **Single Responsibility Principle (SRP) рЃЊрЃљрЃфрЃЋрЃљ** - рЃЊрЃћрЃЎрЃЮрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА рЃфрЃљрЃџрЃЎрЃћрЃБрЃџ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃерЃў рЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃћрЃю.

**Decorator Pattern** PHP-рЃерЃў рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЊрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃџрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃљ **рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА** рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ. рЃЏрЃўрЃАрЃў рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃБрЃарЃљрЃЊ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЊрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃБрЃцрЃарЃЮ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЏрЃљрЃарЃЌрЃЋрЃљ.
