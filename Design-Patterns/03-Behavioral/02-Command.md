**Command Pattern** рЃљрЃарЃўрЃА рЃЦрЃфрЃћрЃЋрЃўрЃЌрЃў рЃбрЃўрЃърЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃ«рЃћрЃџрЃА рЃБрЃгрЃДрЃЮрЃЉрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА (Commands) рЃўрЃюрЃЎрЃљрЃцрЃАрЃБрЃџрЃљрЃфрЃўрЃљрЃА рЃБрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃљрЃЊ. рЃћрЃА рЃърЃљрЃбрЃћрЃарЃюрЃў рЃњрЃљрЃЏрЃЮрЃЊрЃњрЃћрЃЉрЃљ рЃўрЃАрЃћрЃЌ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃЉрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃфрЃљрЃџрЃЎрЃћ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃљрЃЊ рЃљрЃдрЃЌрЃЦрЃЏрЃљ, рЃЏрЃљрЃЌрЃў рЃџрЃЮрЃњрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃљрЃю рЃњрЃљрЃЊрЃљрЃЊрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ.

**Command Pattern** рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ, рЃарЃЮрЃфрЃљ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ undo/redo рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА, рЃерЃћрЃгрЃЋрЃћрЃбрЃљрЃЊрЃў (Queue-bases) рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃљрЃю рЃърЃџрЃљрЃњрЃўрЃюрЃБрЃарЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ.

---

#### рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Command Pattern?

Command Pattern рЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃљ:
1. **Command рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў** - рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃерЃћрЃЏрЃАрЃарЃБрЃџрЃћрЃЉрЃћрЃџ (execute) рЃЏрЃћрЃЌрЃЮрЃЊрЃА.
2. **Concrete Command рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў** - рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА рЃљрЃ«рЃЊрЃћрЃюрЃћрЃю.
3. **Receiver (рЃЏрЃўрЃЏрЃдрЃћрЃЉрЃў)** - рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃА.
4. **Invoker (рЃњрЃљрЃЏрЃЮрЃЏрЃгрЃЋрЃћрЃЋрЃў)** - рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃљ рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃА.
5. **Client (рЃЎрЃџрЃўрЃћрЃюрЃбрЃў)** - рЃЦрЃЏрЃюрЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЏрЃА Invoker - рЃА.
---

#### Command Pattern  - рЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

##### 1. Command рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў
```php
interface Command {
    public function execute(): void;
}
```

##### 2. Receiver (рЃЏрЃўрЃЏрЃдрЃћрЃЉрЃў)

Receiver рЃЎрЃџрЃљрЃАрЃерЃў рЃњрЃљрЃюрЃЋрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃЌ рЃўрЃЏ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃюрЃЊрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА.
```php
class Light {
    public function turnOn() {
        echo "Light is ON\n";
    }

    public function turnOff() {
        echo "Light is OFF\n";
    }
}
```


##### 3. рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў Command рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў

Concrete Command рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА Command рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА рЃЊрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃю Receiver-рЃќрЃћ.
```php
class TurnOnLightCommand implements Command {
    private Light $light;

    public function __construct(Light $light) {
        $this->light = $light;
    }

    public function execute(): void {
        $this->light->turnOn();
    }
}

class TurnOffLightCommand implements Command {
    private Light $light;

    public function __construct(Light $light) {
        $this->light = $light;
    }

    public function execute(): void {
        $this->light->turnOff();
    }
}
```


##### 4. Invoker (рЃњрЃљрЃЏрЃЮрЃЏрЃгрЃЋрЃћрЃЋрЃў)

Invoker рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА Command рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА.

```php
class RemoteControl {
    private Command $command;

    public function setCommand(Command $command): void {
        $this->command = $command;
    }

    public function pressButton(): void {
        $this->command->execute();
    }
}
```

##### 5. Client (рЃЎрЃџрЃўрЃћрЃюрЃбрЃў)

рЃЎрЃџрЃўрЃћрЃюрЃбрЃў рЃЦрЃЏрЃюрЃўрЃА Command рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЏрЃА Invoker-рЃА.

```php
// Receiver рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў
$light = new Light();

// Command рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў
$turnOnCommand = new TurnOnLightCommand($light);
$turnOffCommand = new TurnOffLightCommand($light);

// Invoker рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў
$remote = new RemoteControl();

// рЃњрЃљрЃюрЃљрЃЌрЃћрЃЉрЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ
$remote->setCommand($turnOnCommand);
$remote->pressButton(); // Output: Light is ON

// рЃњрЃљрЃюрЃљрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃарЃЌрЃЋрЃљ
$remote->setCommand($turnOffCommand);
$remote->pressButton(); // Output: Light is OFF
```

#### Command Pattern - рЃўрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў

РюЁ **Loose Coupling (рЃАрЃБрЃАрЃбрЃў рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ)** - рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЎрЃЮрЃЊрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў Receiver-рЃЌрЃљрЃю.    
РюЁ **рЃњрЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ** - рЃерЃћрЃАрЃљрЃФрЃћрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.    
РюЁ**Redo/Undo рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ*** - рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃБрЃЎрЃљрЃю рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ.    
РюЁ **Queue рЃЊрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃюрЃљрЃЎрЃљрЃЊрЃўрЃљрЃюрЃЮрЃЉрЃљ** - рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃљрЃЊрЃЊрЃћрЃА рЃљрЃю Queue рЃерЃў рЃерЃћрЃЋрЃўрЃюрЃљрЃ«рЃЮрЃЌ.

---

#### ­Ъџђ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

Command Pattern рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃФрЃџрЃўрЃћрЃарЃў рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃърЃљрЃбрЃћрЃарЃюрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ **рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃарЃБрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃерЃў, рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃюрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљрЃерЃў** рЃЊрЃљ **рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃўрЃА рЃБрЃцрЃарЃЮ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃљрЃЊ рЃЏрЃљрЃарЃЌрЃЋрЃљрЃерЃў**. рЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЊрЃћрЃњрЃўрЃљ **GUI рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃерЃў, Task Queue рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў** рЃЊрЃљ **рЃърЃџрЃљрЃњрЃўрЃюрЃБрЃа рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃерЃў**.
